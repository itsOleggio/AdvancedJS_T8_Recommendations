# Задание 8.1 Создание витрины магазина с типизацией 

## Описание 

Создание витрины магазина с типизацией, используя технологию [TypeScript](https://www.typescriptlang.org/). 
Основные реализованные фичи:
- Сокращение количество символов в `item.title` если он больше 50
- Расчет цвета карточки товара в зависимости от его количества
- Динамическое выставление символа валюты в щависимости от `item.currency_code`
- Создание типов для карточки товара 

## Код программы и структура проекта

Программа находится в папке `src`.

Структура проекта 
```
AdvancedJS_T8_Recommendations/
├── public/ 
├── data/
│   └── etsy.json // данные о товарах в формате JSON
├── src/
│   ├── components/
│   │   └── Listing/
│   │       └── Listing.tsx // Компонент карточки товара
│   ├── model/
│   │   └── Card.model.ts // модель карточки товара
│   ├── utils/
│   │   └── getStockClass.ts // вспомогательные функции для расчета цвета карточки
│   ├── App.jsx // компонент приложения
│   ├── App.css // общий стиль для всего приложения
│   |── index.css // корневые стили для проекта 
│   └── main.jsx // главный компонент приложения
├── index.html
├── package.json
└── vite.config.js
```

## Запуск программы

- Сборка программы: `npm run build`
- Запуск программы: `npm run dev`
- Привью программы: `npm run preview`
